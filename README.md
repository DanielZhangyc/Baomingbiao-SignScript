# Baomingbiao-SignScript

一个基于Streamlit的报名表自动提交Web应用，专门针对报名表小程序设计的抢课工具。

## 特点

- 🚀 自动提交表单
- ⏰ 定时提交功能
- 📝 动态字段管理
- 📊 实时提交状态显示
- 💾 默认配置保存

## 安装

```bash
# 克隆仓库
git clone https://github.com/yourusername/Baomingbiao-SignScript.git

# 安装依赖
pip install -r requirements.txt
```

## 使用

```bash
streamlit run app.py
```

## 注意事项

- Token抓包获取
